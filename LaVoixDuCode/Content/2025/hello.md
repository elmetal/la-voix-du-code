# はじめまして

**La voix du code** へようこそ。これは *Ignite* でビルドされ、**GitHub Pages** に自動デプロイされる Markdown 記事のサンプルです。

- 記事は `Content/` 配下に Markdown で追加します。
- `main` に push すると GitHub Actions が `ignite build` を実行して `Build/` を Pages に公開します。

Bonne écriture! ✨
